HTML

HyperText Markup Language

HTML (Linguagem de Marcação de Hipertexto) é a linguagem base para criar o conteúdo e a estrutura das páginas web. É utilizada para definir elementos como parágrafos, cabeçalhos, imagens e links, indicando aos navegadores como exibir esses elementos .

  Apesar de ser uma linguagem, O HTML não é uma linguagem de programação ,mas sim uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante. Em outras palavras, o HTML define o "esqueleto" de um site, enquanto outras tecnologias como CSS e JavaScript cuidam da aparência e da funcionalidade.

Títulos - h1 ao h6

Para criar o titulo acima foi preciso declarar uma tag " h1/> ", para este texto aqui, declarei a tag "p/>.
E para vir aqui para baixo, utilizei a tag "br/>". Mas qual o objetivo dessas tags afinal? Atributos HTML (em inglês html attributes") são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles, podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc

Listas - Ordernadas & Não Ordenadas

O elemento HTML < ul> (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. Já o elemento HTML < ol> (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados.

EXEMPLOS DE LISTAS:

Lista Desordenada de Compras:

  • Leite
  • Ovos
  • Farinha

Lista Ordenada de Corredores:

  1. Ferrari
  2. Skyline
  3. Golf
